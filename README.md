## Quick start
```
# start android
yarn install
react-native run-android
```

## Appendix
- create a react-native project
```
npm install -g react-native-cli yarn
npm update -g react-native-cli
react-native init MyApp
cd MyApp
yarn install
```
- run a project
```
react-native run-android
react-native run-ios
```
